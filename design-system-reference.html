06 — GSAP Animation Patterns

Expressive scroll-triggered animations using GSAP + ScrollTrigger. These are the recommended patterns for each section type.

Hero Reveal

Staggered text lines + fade-in image overlay on page load.

gsap.from('.hero-line', { y: 80, opacity: 0, duration: 1.2, ease: 'power3.out', stagger: 0.15, delay: 0.3 })

Section Fade Up

Content blocks slide up and fade in as they enter viewport.

gsap.from(el, { y: 60, opacity: 0, duration: 1, ease: 'power2.out', scrollTrigger: { trigger: el, start: 'top 85%' } })

Process Flow

Steps reveal with organic timing — image scales, text slides.

gsap.from('.step-img', { scale: 1.15, opacity: 0, duration: 1.2, ease: 'power2.out', scrollTrigger: { trigger: '.step', start: 'top 75%' } })

Parallax Image

Background images move slower than scroll for depth.

gsap.to('.parallax-img', { yPercent: -20, ease: 'none', scrollTrigger: { trigger: '.parallax', start: 'top bottom', end: 'bottom top', scrub: true } })

Text Split Reveal

Big headlines reveal word-by-word or line-by-line.

// Split text into words gsap.from('.word', { y: '100%', opacity: 0, duration: 0.8, ease: 'power3.out', stagger: 0.04, scrollTrigger: { trigger: '.headline', start: 'top 80%' } })

Line Reveal

Decorative lines and dividers animate their width.

gsap.from('.line', { scaleX: 0, transformOrigin: 'left', duration: 1, ease: 'power2.inOut', scrollTrigger: { trigger: '.line', start: 'top 85%' } })

Our Story

From Garden to Cup

At Bagaicha Bliss, every journey begins long before it reaches your hands. It starts in sun-kissed coffee gardens where rich soil, fresh air, and careful cultivation shape the character of each bean. Here, patience is a virtue — cherries are allowed to ripen slowly, absorbing the best that nature has to offer.

The Journey

Crafting Coffee, Step by Step

01

Hand-Picking the Cherries

High in Nuwakot's hills, our journey starts in carefully nurtured coffee gardens. Each plant is grown with patience, care, and respect for the natural ecosystem that makes the region unique.

02

Wet Processing

After harvesting, the beans are carefully processed in small batches. The unique wet-processing method removes the fruit and allows the beans' natural flavors to develop clarity and depth.

03

Natural Drying

Spread across raised drying beds under the Himalayan sun, the beans dry slowly and naturally. This patient process preserves the complex flavors that make our coffee distinctive.

04

Roasted with Intention

Each batch is roasted in small quantities to unlock the full spectrum of flavor. Timing, temperature, and technique combine to create a balanced, rich profile unique to Nuwakot beans.

09 — Usage Notes
Design Principles
Always warm, never cold

Every shadow, border, and background should use warm tones from the coffee palette. Never use grey (#ccc, #999), use coffee-200, coffee-300 instead. Never use pure black (#000), use coffee-900 or coffee-950.

Breathe generously

Use the full section spacing tokens. Premium brands need room to breathe. If a section feels cramped, add more space. The rhythm of the scroll is part of the storytelling.

Sharp edges, soft textures

Buttons and cards use no border-radius (or very minimal). But backgrounds use grain texture, warm gradients, and organic photography. The contrast between sharp geometry and natural imagery creates premium tension.

Text hierarchy is everything

Always lead with an overline accent, then the heading, then body text. This creates a consistent visual rhythm across all sections. The overline with the horizontal line is your signature element.

Grain is your friend

The subtle grain overlay adds organic texture to the entire page. It prevents flat digital backgrounds from looking sterile. Keep it subtle (opacity ~0.35).
